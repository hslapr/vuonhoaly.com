論理回路設計

位取り記数法 p 進数 N = (dn−1dn−2…d1d0.d−1d−2…d−m)p = dn−1·pn−1 + dn−2·pn−2 + … + d1·p1 + d0 + d−1·p−1 + d−2·p−2 + … + d−m·p−m （括弧と添字 p は基数 p = 10 のとき省略）

q 進数を p 進数に変換： q 進数を 10 進数に変換し，その 10 進数を p 進数で表す。

10 進数 ⇒ p 進数：整数部と小数部を分けて変換する。

整数の場合の例：(58)10 を 3 進数に変換
58 ÷ 3 = 19 … 1 ⟹ 19 ÷ 3 = 6 … 1 ⟹ 6 ÷ 3 = 2 … 0
ゆえに， (58)10 = (2011)3

小数の場合の例：(0.8)10 を 3進数に変換
0.8 × 3 = 2.4 ⟹ 0.4 × 3 = 1.2 ⟹ 0.2 × 3 = 0.6 ⟹ 0.6 × 3 = 1.8 ⟹ 0.8 × 3 = 2.4 … ゆえに，(0.8)10 = (0.2˙101˙)3 （点は循環小数を表す）

2 進数の 3 桁分および 4 桁分はそれぞれ 8 進数および 16 進数の 1 桁分に相当する。

情報の単位
ビットbit2 進数 1 桁分
バイトbyte8 ビット
ワード / 語wordコンピュータにおける処理の単位
MSB1 ワードの最上位ビット
LSB1 ワードの最下位ビット

2 進数を定められたビット数を用いて表す方式の中で，小数点の位置をあらかじめ決めておくものを固定小数点表示といい，この方式で表された数を固定小数点数（fixed point number）という。

2 進数 N の 1 の補数（1's complement） NC1 は N の各桁を反転した数で， 2 の補数（2's complement）NC2 は NC1 の LSB に 1 を加算した数。

固定小数点表示において負の数を表す方法：

符号絶対値表現（sign-magnitude representation）
MSB を符号ビットとして用い，1 ならば負，0 ならば非負の数を表す。
バイアス（ゲタ履き）方式表現（biased number representation）
一律に同じ値（バイアス）を加える。
2 の補数表現（2's complement representation）
負の数はその絶対値の 2 の補数で表す。 MSB は符号絶対値表現と同じく符号ビット。

固定小数点数の減算：数 N を減算する代わりに， NC2 を加算。 N − M ⇒ N + MC2 = N − M + 2n 。

情報の表現と演算

位取り記数法 p 進数 N = (dn−1dn−2…d1d0.d−1d−2…d−m)p = dn−1·pn−1 + dn−2·pn−2 + … + d1·p1 + d0 + d−1·p−1 + d−2·p−2 + … + d−m·p−m （括弧と添字 p は基数 p = 10 のとき省略）

q 進数を p 進数に変換： q 進数を 10 進数に変換し，その 10 進数を p 進数で表す。

10 進数 ⇒ p 進数：整数部と小数部を分けて変換する。

整数の場合の例：(58)10 を 3 進数に変換
58 ÷ 3 = 19 … 1 ⟹ 19 ÷ 3 = 6 … 1 ⟹ 6 ÷ 3 = 2 … 0
ゆえに， (58)10 = (2011)3

小数の場合の例：(0.8)10 を 3進数に変換
0.8 × 3 = 2.4 ⟹ 0.4 × 3 = 1.2 ⟹ 0.2 × 3 = 0.6 ⟹ 0.6 × 3 = 1.8 ⟹ 0.8 × 3 = 2.4 … ゆえに，(0.8)10 = (0.2˙101˙)3 （点は循環小数を表す）

2 進数の 3 桁分および 4 桁分はそれぞれ 8 進数および 16 進数の 1 桁分に相当する。

2 進数を定められたビット数を用いて表す方式の中で，小数点の位置をあらかじめ決めておくものを固定小数点表示といい，この方式で表された数を固定小数点数（fixed point number）という。

2 進数 N の 1 の補数（1's complement） NC1 は N の各桁を反転した数で， 2 の補数（2's complement）NC2 は NC1 の LSB に 1 を加算した数。

固定小数点表示において負の数を表す方法：

符号絶対値表現（sign-magnitude representation）
MSB を符号ビットとして用い，1 ならば負，0 ならば非負の数を表す。
バイアス（ゲタ履き）方式表現（biased number representation）
一律に同じ値（バイアス）を加える。
2 の補数表現（2's complement representation）
負の数はその絶対値の 2 の補数で表す。 MSB は符号絶対値表現と同じく符号ビット。

固定小数点数の減算：数 N を減算する代わりに， NC2 を加算。 N − M ⇒ N + MC2 = N − M + 2n 。

論理演算と論理関数

命題（proposition, statement）とは，内容の真偽が決定できる文（文章）。真（true, 1）と偽（false, 0）2 値のいずれかを取る。

変数 x を含み， x が定まれば真偽が決定できるような命題を命題関数といい，変数 x の定義域を命題関数の対象領域という。

対象領域のすべての x に対して真となるような命題は恒真命題といい，すべての x に対して偽となるような命題は恒偽命題という。

命題を接続詞で結合したり，否定したりすることによって作られる命題を複合命題という。

集合と命題： S1 = {x ∈ U ∣ P1(x) が真} ， S2 = {x ∈ U ∣ P2(x) が真} とするとき， P1(x) ∧ P2(x) を真とする集合は S1 ∩ S2 ， P1(x) ∨ P2(x) を真とする集合は S1 ∪ S2

P1P2P1→P2P1⊕P2
偽偽真偽
偽真真真
真偽偽真
真真真偽

P1(x) → P2(x) を真とする集合は S1 ∪ (S1 ∩ S2) ， P1(x) ⊕ P2(x) を真とする集合は (S1 ∩ S2) ∪ (S1 ∩ S2)

ならば（→）：前半が偽であれば後半に関係なく命題は真になる。

論理値 0 あるいは 1 のどちらかの値をとる変数を（2 値の）論理変数（logical variable）といい， x を論理変数 x の否定形， x 自身を x の肯定形という。 x, x を総称してリテラル（literal ，文字）という。

論理値（定数）および論理変数に論理演算を任意の順序で任意の回数施して得られる式を論理式（logical expression）という。

優先度は · > ⊕ = + 。 · は省略可。

論理関数は， Bn から B への写像（B = {0, 1}）。真理値表によって一意的に定義できる。論理式は，変数と演算の組み合わせによって定義される。一般に，一つの論理関数に対して複数の等価な論理式が存在する。

真理値表が同じであれば， 2 つの論理式が等価という。

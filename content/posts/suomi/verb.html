芬兰语有 4 种时，其中 2 种（现在时、过去时）是简单时态， 2 种（现在完成时、过去完成时）是复合时态。复合时态的助动词都是 olla 。

芬兰语有 4 种语气，分别是：直陈语气（indicative）、条件语气（conditional）、命令语气（imperative）、可能语气（potential）。

动词的构成方式是：词干 [+ 过去时后缀] [+ 人称后缀]

过去时

过去时用后缀 -i 表示，放在词干之后，人称词尾之前。

过去时的否定是否定词 + “NUT 分词”。

现在完成时

现在完成时由助动词 olla 的现在时 + “NUT 分词”组成。“NUT 分词”是由动词词干 + -nut / -nyt 构成。如果词干以辅音 l/r/s 结尾，则词缀的 n 被同化；如果词干以辅音 t 结尾，则那个 t 被同化为 n 。

现在完成时的否定是否定词 + ole + “NUT 分词”。

过去完成时

过去完成时由助动词 olla 的过去时 + “NUT 分词”组成。

过去完成时的否定是否定词 + ollut / olleet + “NUT 分词”。

条件语气

条件语气用后缀 -isi 表示，放在词干之后，人称词尾之前。否定是否定词 + 不带人称后缀的动词条件语气。

可能语气

可能语气用后缀 -ne 表示，放在词干之后，人称词尾之前。如果词干以辅音 l/r/s 结尾，则词缀的 n 被同化；如果词干以辅音 t 结尾，则那个 t 被同化为 n 。

可能语气现在时否定是否定词 + 不带人称后缀的动词可能语气，完成时的否定是否定词 + olla 的可能语气 + NUT 分词

动词的不定式

芬兰语的动词有 5 种不定式，不定式具有一定的名词性质，可以接格后缀。第一不定式也叫“A 不定式”，以 [dtn𝕃]?𝕒 结尾。第一不定式可以接转移格后缀。第二不定式也叫“E 不定式”，就是把第一不定式的 𝕒 换成 e。第二不定式可以接内格和手段格后缀。第三不定式也叫“MA 不定式”，以 -m𝕒 结尾。第三不定式可以接内格、出格、入格、接格、欠格、手段格后缀。第四不定式也叫“MINEN 不定式”，后缀是 minen 。第四不定式有主格和部分格，但很少用。第四不定式常常作为动名词使用。

动词的分词

芬兰语的动词有 2 种分词，分别是现在分词和过去分词，各有主动和被动两种形式。

NUT 分词是形容词性的，也就是说有数和格，接词缀时的形式是 -nee ，比如复数时是 -neet 。

动词的被动态

芬兰语动词的被动态现在时由第一不定式 + -𝕍n 构成。被动态过去时由词干 + -tiin / -ttiin 构成。被动过去分词就是被动态过去时的 -iin 换成 𝕦 。被动态的现在完成时和过去完成时由 on / oli + 被动过去分词构成。

被动态的各种语气（没有直陈语气）是把被动过去时的 -iin 换成 𝕒 再加上语气的后缀然后再加上 -𝕍n 。

否定

现在时的否定是由否定词 ei + 人称后缀 + 动词现在时不带人称后缀的形式。

Help

Why learn through sentences?

It is much more efficient than memorizing word lists.

Supported languages

All languages using Latin script. Accent symbols are allowed.

How to use?

Prepare the data file

Prepare a plain text file containing sentences that you want to learn. Here is a sample file.

The first line and each line with a preceding blank line are treated as valid sentences.
If a sentence contains newline, each line should end with a backslash "\" except the last line.

Load the file

Drag & drop your file to the application page.

Start learning

You can set the number of blanks. 0 means to show the original sentence. If you want to change the blank number of the current sentence, click "again" button after you change the number.

Press Ctrl+Enter while the cursor is in a blank or click "check" button to check your answer.

Pressing Tab key to move to the next blank. Press Shift+Tab to move to the last one.

When the cursor is not in a blank, press →/← to go to the next/previous sentence.

When the cursor is not in a blank, press ↑/↓ to add/reduce a blank and refresh. Press Enter to refresh without changing the number of blanks.
